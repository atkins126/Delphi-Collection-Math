# CalculSum
- A Simple Way To Write Addition Operations Using Object Oriented Programming(OOPs).                                                              
![](CalculSum.png) 



# Features  
- Implimentation Of Design Pattern That Allows You To Think About Problems In Terms Of Objects and Their Interactions. .
